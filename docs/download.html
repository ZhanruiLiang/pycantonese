Download and installÂ¶

PyCantonese is compatible with both Python 2 and 3 (specifically, 2.7 and 3.4+). It is available via pip:

$ pip install pycantonese

To test your installation in the Python interpreter:

>>> import pycantonese
>>> pycantonese.__version__  # show version number
